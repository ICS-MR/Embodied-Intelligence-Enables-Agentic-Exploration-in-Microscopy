## 3. Expected Results

The expected outcome is to complete the low-magnification localization and high-magnification acquisition workflow for organoid samples: first, use a 4× objective to detect and determine the positions of all organoid regions, record the corresponding coordinates for subsequent acquisition, then switch to a 60× objective to perform detailed imaging of each detected organoid region. High-magnification acquisition should cover the respective organoid locations and incorporate the three-dimensional structure of the organoid for Z-stack data acquisition.

## 4. Execution result and failure cause analysis

Execution result: The workflow switched to 4× bright-field overview acquisition and detected organoids, then switched to 60× and configured batch acquisition and Z-stack for detected targets.
Failure analysis: The core workflow includes 4× localization, 60× acquisition, and organoid Z-stack consideration, but the code directly uses the return value of `analysis_platform_find_target_positions()` as physical stage coordinates for 60× acquisition; this direct return value should be treated as image pixel bounding-box coordinates, and it was not loaded or converted into physical center coordinates usable for stage movement, so the 60× detailed acquisition likely did not land on the corresponding organoid region. Additionally, the risk that `organoid_positions[0]` will fail when there are no detection results was not handled.

say("[INFO] Starting organoid detection and detailed imaging workflow")

say("[ACTION] Setting up 4x objective for organoid detection")
target_obj_4x = '1-UPLFLN4XPH'
current_obj = get_objective()
if current_obj != target_obj_4x:
    set_objective(target_obj_4x)
    say(f"[INFO] Objective switched to 4x (label: {target_obj_4x})")
else:
    say(f"[INFO] Objective is already 4x (label: {target_obj_4x}), no change needed")

say("[ACTION] Configuring brightfield mode for organoid detection")
target_channel = '1-NONE'
current_channel = get_channel()
if current_channel != target_channel:
    set_channel(target_channel)
    say(f"[INFO] Filter set to brightfield mode (channel: {target_channel})")
else:
    say(f"[INFO] Filter is already in brightfield mode (channel: {target_channel}), no change needed")

say("[ACTION] Setting appropriate exposure for brightfield imaging")
target_exposure_bright = 10.0
set_exposure(target_exposure_bright)
say(f"[INFO] Exposure time set to {target_exposure_bright} ms for brightfield")

say("[ACTION] Performing automatic brightness adjustment")
optimal_brightness = perform_autobrightness()
set_brightness(optimal_brightness)
say(f"[INFO] Brightness automatically adjusted to {optimal_brightness}")

say("[ACTION] Performing autofocus for clear organoid detection")
optimal_z = perform_autofocus()
set_z_position(optimal_z)
say(f"[INFO] Autofocus completed, Z position: {optimal_z} μm")

say("[ACTION] Capturing overview image for organoid detection")
current_x, current_y = get_x_y_position()
add_acquisition_position(name="organoid_overview_4x", x=current_x, y=current_y, width=None, height=None)
add_channels(channel=target_channel, exposure=target_exposure_bright)
run_acquisition()
say("[INFO] Overview image captured at 4x magnification")

say("[ACTION] Detecting organoid positions in the captured image")
fiji_initialize()
overview_image = load_image("organoid_overview_4x.ome.tif")
organoid_positions = analysis_platform_find_target_positions(overview_image, "organoid", "Detected organoid positions from 4x overview")
fiji_shutdown()
say(f"[INFO] Detected {len(organoid_positions)} organoid regions")

say("[ACTION] Setting up 60x objective for detailed organoid imaging")
target_obj_60x = '5-LUCPLFLN60X'
set_objective(target_obj_60x)
say(f"[INFO] Objective switched to 60x (label: {target_obj_60x})")

say("[ACTION] Adjusting parameters for 60x imaging")
target_exposure_60x = 15.0
set_exposure(target_exposure_60x)
say(f"[INFO] Exposure adjusted to {target_exposure_60x} ms for 60x imaging")

optimal_brightness_60x = perform_autobrightness()
set_brightness(optimal_brightness_60x)
say(f"[INFO] Brightness re-adjusted for 60x: {optimal_brightness_60x}")

say("[ACTION] Configuring detailed acquisition for all detected organoids")
for i, (center_x, center_y, width, height) in enumerate(organoid_positions):
    organoid_name = f"organoid_{i+1}_60x"
    add_acquisition_position(name=organoid_name, x=center_x, y=center_y, width=width, height=height)
    say(f"[INFO] Added position {i+1}: {organoid_name} at ({center_x}, {center_y}) μm")

add_channels(channel=target_channel, exposure=target_exposure_60x)

say("[ACTION] Determining Z-stack parameters for 3D organoid imaging")
set_x_y_position(organoid_positions[0][0], organoid_positions[0][1])
optimal_z_60x = perform_autofocus()
set_z_position(optimal_z_60x)
z_max, z_min = z_stack_range()
z_step = 0.5
set_z_stack(z_start=z_min, z_end=z_max, z_step=z_step)
say(f"[INFO] Z-stack configured: {z_min} to {z_max} μm with {z_step} μm steps")

say("[ACTION] Starting detailed 60x acquisition of all organoid regions")
run_acquisition()
say("[INFO] Detailed organoid imaging completed successfully")
