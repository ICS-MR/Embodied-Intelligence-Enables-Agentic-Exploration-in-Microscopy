Autofocus should first be performed in the target organoid field of view, followed by Z-stack acquisition covering the thickness range of the organoid. The expected output should include the saved organoid Z‑stack image sequence and its composited sharp image, for subsequent acceptance review.

## 4. Execution Results and Failure Cause Analysis

Execution Results: The workflow first autofocuses on the current organoid field and recommends a Z-stack range, then acquires a brightfield Z-stack, and performs maximum intensity Z projection on the resulting OME-TIFF to generate a sharp composite image.
Failure analysis: The core objectives — Z-axis imaging, autofocus followed by Z-stack recommendation, configuration of Z-stack acquisition, and calling z_projection to save a sharp composite — have all been completed. However, in brightfield as the core acquisition mode, perform_autobrightness() or an equivalent automatic brightness optimization was not executed; the process relied solely on the initial brightness. This poses risks to brightfield image quality and the reliability of the subsequent sharp composite, and therefore it should not be judged as fully successful.

Overall assessment: Partial success
